VR Discover Modes

One of the most frequent tasks was to Showcase some products in VR.
The worksteps to do so were the following:

    1. Reduce the vertex count since they had to run standalone on Quest 2/Quest 3
    2. Merge Parts to make the material assignment faster
    3. Create and Assign fitting Materials in Unity
    4. Create new visualization functionalities if needed
I also restructered the whole way those visualization functions work, since this was all one big script beforehand.

Spacial anchor System for AR Layoutplanning

Shortly after the release of the Meta Quest 3, I developed a custom spatial anchor system. The Quest 3 offers the opportunity to generate a mesh based on the room you are currently in.
Based on this mesh, I created a system where you can select a specific corner as the anchor.
(An anchor is a virtual reference point in the real world used to postion and lock digital objects in a fixed location)
This proved to be very useful for planning layouts of production halls, as there is always precise data about the room.
The models shown in the video are an actual layout as it was planned for a warehouse.
The real space in which I recorded this is, of course, not the warehouse, but an empty office, so the layout doesn't align here.
